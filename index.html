About

Fantasy Sports API is a easy, free-to-use API that allows developers and fantasy sports players to query up-to-date players and games data. The stats are returned in a JSON format for future processing and analysis, which can be easily built upon. A variety of queries are available to query NBA, NHL, and MLB stats.

Fantasy Sports API is currently the only free-to-use API available on the web. Whether you're a developer wanting to build applications on top of sports data easily and free of cost, or an active fantasy sports player looking to take the next step to ensure victory, Fantasy Sports API is for you.

Tutorial

1) Obtain user token.

Click on the above button to generate token to be able to access the queries. These tokens can be used forever once generated.

2) Submit custom SQL query, or use our built-in predefined queries via GET methods, with the definitions shown below.


In order to call our functions, Http Get/Post requests are needed to send/receive the data. This tutorial will provide sample usage of our API in Java using the Jersey http client plugin to formulate the web requests.

Download Jersey from https://jersey.java.net/

The API is hosted at the following url http://52.25.54.103:8080/FantasySports/

{host}/query/getPlayerCurrentOverallStats/{playerName}/{league}/{token}

This query gets the player's overall stats for the entire season up until today's date. playerName is a player's full name, and league is an enum of {NBA, MLB, NHL}.



{host}/query/getPlayerDailyStats/{playerName}/{league}/{date}/{token}

This query gets the player's stats for a single date provided. playerName is a player's full name, league is an enum of {NBA, MLB, NHL}, and date is in a YYYY-MM-DD format.


The following is an example of how to write a query where we want to get Stephen Curry's stats for 2015-10-27.

private static void getPlayerDailyStats() throws Exception{
    String baseURL = "http://52.25.54.103:8080/FantasySports/query/getPlayerDailyStats/";
    String playerName = java.net.URLEncoder.encode("Stephen Curry", "UTF-8").replace("+", "%20");
    String league = "NBA";
    String date = "2015-10-27";
    String token = "3a2993df-e8a2-11e5-9f31-0228a770e05b";
    URL url = new URL(baseURL + playerName + "/" + league + "/" + date + "/" + token);

    Client client = Client.create();
    WebResource webResource = client
            .resource(url.toString());
    String response = webResource.accept("application/json")
            .get(String.class);
    JSONArray ja = new JSONArray(response);
    String csv = CDL.toString(ja);
    System.out.println(csv);
}


{host}/query/getPlayerPeriodicStats/{playerName}/{league}/{startDate}/{endDate}/{token}

This query gets the player's stats for the period provided, inclusive. playerName is a player's full name, league is an enum of {NBA, MLB, NHL}, startDate and endDate are in a YYYY-MM-DD format.



{host}/query/getTeamCurrentOverallStats/{teamName}/{league}/{token}

This query gets the team's overall stats for the entire season up until today's date. teamName is a team's full name, and league is an enum of {NBA, MLB, NHL}.



{host}/query/getTeamDailyStats/{teamName}/{league}/{date}/{token}

This query gets the team's stats for a single date provided. teamName is a team's full name, league is an enum of {NBA, MLB, NHL}, and date is in a YYYY-MM-DD format.

This query gets the team's stats for the period provided, inclusive. teamName is a player's full name, league is an enum of {NBA, MLB, NHL}, startDate and endDate are in a YYYY-MM-DD format.



{host}/query/getNBAAdvancedStats

This query allows the user to send a POST command with a JSON string as the argument, allowing more flexibility to the NBA results.


The following is an example of how to write a query where we want a list of games where Stephen Curry has at least 30 points and 5 assists. 

private static void getNBAAdvancedStats() throws Exception {
    Client client = Client.create();

    WebResource webResource = client
            .resource("http://52.25.54.103:8080/FantasySports/query/getNBAAdvancedStats");

    String input = "{\"playerName\":\"Stephen Curry\", \"PTS\":\"30\", \"AST\":\"5\",\"token\":\"3a2993df-e8a2-11e5-9f31-0228a770e05b\"}";

    String response = webResource.type("application/json")
            .post(String.class, input);
    JSONArray ja = new JSONArray(response);
    String csv = CDL.toString(ja);
    System.out.println(csv);
}

{host}/query/getCustomQuery

This query allows the user to send a POST command with a custom SQL string, allowing advanced user for utmost flexibility in what they want to retrieve. The database schema is provided below for reference. The SQL language used is MySQL. In order to submit a SQL string for the API to process, it must be in the following form

{query: SQL query, token: token}

This is the json format necessary to submit the SQL query for process.



The following is an example of how to write a custom query where we want a list of games where Stephen Curry hits at least ten 3 pointers in a game.

private static void query2() throws Exception{
    Client client = Client.create();

    WebResource webResource = client
            .resource("http://52.25.54.103:8080/FantasySports/query/getCustomQuery");

    String input = "{\"query\":\"Select * from NBAPlayerStats as a, NBAPlayer as b where playerName = 'Stephen Curry' and `3P` >= 10 and a.playerID = b.playerID\",\"token\":\"3a2993df-e8a2-11e5-9f31-0228a770e05b\"}";

    String response = webResource.type("application/json")
            .post(String.class, input);
    JSONArray ja = new JSONArray(response);
    String csv = CDL.toString(ja);
    System.out.println(csv);
}
